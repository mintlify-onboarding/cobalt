You can find your Monday.com app credentials by visiting your Twilio account.

You'll need the following information to set up your Twilio.com app with Cobalt:

* Account SID

* API Key

* Twilio Number

### Getting your Twilio.com API Key

1. 1.  
Go to your Twilio.com account.

2. 2.  
Click the Account icon in the bottom-left corner

3. 3.  
Select **Admin**

4. 4.  
Select "API" from the left sidebar.

5. 5.  
Copy the API Key under **Personal API Token**.
